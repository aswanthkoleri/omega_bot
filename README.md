# Welcome to Omega-Bot!
## Team Chunkzz
**Omega-Bot**  is an a zulip chatbot   


# Features

Omega-Bot can :
>
>* Check PNR status
>* Check Crypto Currency
>* Must-read functionality for channels
>*  Motivates you with quotes
>* Send mails via zulipchat
>* Shorten URL's
>* Shows weather
>* Use Gif while chatting :wink:
